# SACHHSOFT — Official Website

Bespoke software development studio. Chicago & Global.

Built with **Next.js 15**, **Tailwind CSS v4**, and **Framer Motion**.

---

## Pages

| Route | Description |
|---|---|
| `/` | Homepage — Hero, Services, Process, Reach & Impact, Testimonial, Industries, CTA |
| `/about` | About — Story, Values, Team, Stats |
| `/services` | Services — Product Design, Cloud/DevOps, AI, Team Augmentation |
| `/blog` | Blog — Live articles from Medium RSS feed |
| `/start-a-project` | Multi-step project intake form |
| `/industries` | Industries served *(in progress)* |
| `/case-studies` | Case studies *(in progress)* |

---

## Tech Stack

- **Framework** — Next.js 15 (App Router, SSR/SSG)
- **Styling** — Tailwind CSS v4
- **Animations** — Framer Motion
- **Icons** — Lucide React
- **Fonts** — Cormorant (serif) + Outfit (sans-serif) via Google Fonts
- **Blog** — Medium RSS via rss2json API
- **Package Manager** — pnpm

---

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Design Tokens

| Token | Value | Usage |
|---|---|---|
| `ink` | `#18160f` | Primary dark background & text |
| `gold` | `#b8975a` | Brand accent |
| `gold-lt` | `#d4b47e` | Lighter gold for italic highlights |
| `off` | `#f8f7f5` | Off-white section backgrounds |
| `rule` | `#e2dfd8` | Borders and dividers |
